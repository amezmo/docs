# Environments

Environments are associated with an [instance](../instances/index.md), and a Git repository. You can define deployment rules for environments
and setup automatic deployments with an API based deployment with an archive file, or a Git provider deployment using GitHub, BitBucket, or GitLab.

- `GET` [List environments](list-environments.md)
- `GET` [Get an environment](get-environment.md)
- `PATCH` [Update an environment](update-environment.md)

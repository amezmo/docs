# Get a region

`GET` /v1/regions/{region_id}

## Parameters

Parameter | Type | In | Description
--------- | ---- | -- | -----------
region_id | string | uri | The region ID.

## Response

`200 OK`

{title="200 OK"}
```json
{
    "id": "au2-au",
    "iso_country_code": "AU",
    "name": "Australia"
}
```

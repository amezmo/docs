# Terminating

Terminating, or deleting, an application removes it from Amezmo completely.
Deleting an application is permanent and cannot be undone.

If you only need to take an instance offline temporarily,
[pause the instance](pausing.md) instead. Pausing keeps your data and can be
reversed.

> [!IMPORTANT]
> As soon as you terminate an application, you stop being charged for it.

When you delete an application, your final charge is rounded up to the end of
the current hour. For example, if you delete an application at 5:15 PM, you are
charged until 6:00 PM. You pay for usage by the hour, rounded up to the nearest
full hour.

Terminating can take a few minutes to finish. Amezmo uses the timestamp of when
you started the termination to set your final charge, so if you terminate at
5:58 PM and Amezmo takes a while to process it, you are still only charged until
5:58 PM.
